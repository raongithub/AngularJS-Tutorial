AngularJS Filters

    * Ant AngularJS filter usually takes input and gives output
    * Can be used to
        > Format data
        > Transform data
        > Search/filter data
    * Can be used in
        > Templates (views)
        > Controllers
        > Service
    * Types of filters
        > Built-in (comes with AngularJS)
            >> number, date, filter, uppercase, lowercase, orderBy etc.
        > Custom
            >> developed by us
